Changing Proper Case Of A String in "C"

To change the proper case in string i.e. if the character of string is in lower case then it is converted to the upper case and if the character is in upper case then it is converted to the lower case.

example:

input : iNdIa;

output : InDiA;

  • Take a string of any length.
  • scan string character
  • If the character index is in the upper case ,then add 32 to convert it in lower case ,else substract 32 to convert in lower case.
